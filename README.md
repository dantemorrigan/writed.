# Writed.

Минималистичный редактор для писателей. Бумага и чернила, швейцарская типографика, никаких отвлечений.

## Запуск локально

Это статичное приложение (React + Babel грузятся из CDN, JSX транспилируется в браузере). Откройте через любой локальный сервер:

```bash
# из этой папки
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

Открывать `index.html` напрямую через `file://` не получится — браузер блокирует загрузку модулей. Нужен http-сервер.

## Деплой на GitHub Pages

1. Запушьте содержимое этой папки в ветку `main`.
2. В репозитории: **Settings → Pages → Build and deployment → Source: Deploy from a branch**, ветка `main`, папка `/ (root)`.
3. Через минуту сайт будет доступен по адресу `https://dantemorrigan.github.io/writed./`.

Файл `.nojekyll` нужен, чтобы GitHub Pages отдавал `.jsx`-файлы без обработки Jekyll.

## Структура

| Файл | Назначение |
|------|-----------|
| `index.html` | Точка входа |
| `styles.css`, `ui.css` | Дизайн-система и интерфейс |
| `store.js` | Локальное хранилище (localStorage) |
| `app.jsx` | Корень приложения и роутинг |
| `components.jsx` | Иконки и общие компоненты |
| `screens-*.jsx` | Экраны: онбординг, дашборд, редактор, экспорт, профиль |
| `paper.svg` | Текстура бумаги |
