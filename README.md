# Writed.

Минималистичный редактор для писателей — бумага и чернила, швейцарская типографика, ноль отвлечений.

**→ [writed. live](https://dantemorrigan.github.io/writed./)**

---

## Что это

Writed. — локальное веб-приложение для тех, кто пишет книги, рассказы и заметки. Всё хранится только на вашем устройстве. Никакой регистрации, никакого сервера, никаких подписок.

Построено на принципах iA Writer: одна колонка, выверенный шрифт, всё лишнее убрано.

---

## Возможности

### Письмо
- **Проекты и главы** — организуйте книги в проекты, главы перетаскиваются мышью
- **Заметки** — отдельный поток для идей, эпиграфов, черновиков
- **Богатое форматирование** — жирный, курсив, зачёркнутый, подчёркнутый, заголовки H1–H3, цитаты, списки, разделители
- **Фокус-режим** — полноэкранный режим без интерфейса, выход по Esc
- **Автосохранение** — debounced через 500 мс после последнего нажатия

### Шрифты прямо в редакторе
Шрифт переключается в тулбаре редактора — не нужно выходить в настройки. Три варианта:
- **Newsreader** — классическая книжная антиква
- **Spectral** — чистый, современный шрифт для статей
- **JetBrains Mono** — моноширинный для сценариев и технических текстов

### Превью и экспорт
- **Превью книги** — переключение между режимами "Редактор" и "Превью" прямо в шапке
- **Экспорт** — гибкая настройка: выбор глав, шрифт, поля, нумерация страниц, оглавление, титульный лист

### Профиль
- **Аватар** — загрузите фото, оно автоматически обрезается до 400×400 px и отображается в шапке сайта
- **Имя** — задаётся при онбординге, редактируется в профиле
- **Тема** — светлая (тёплая бумага) или тёмная (ночное письмо)
- **Статистика** — слов написано, проектов, глав, заметок

### Данные
- **Локальное хранение** — всё в `localStorage`, ничего не уходит на сервер
- **Бэкап / восстановление** — экспорт всего в JSON и импорт обратно
- **Сброс** — возврат к состоянию по умолчанию с подтверждением

## Дизайн

| Элемент | Решение |
|---|---|
| Типографика | Newsreader, Spectral, JetBrains Mono — Google Fonts |
| Палитра (светлая) | Тёплая бумага `#f4f1ea`, чернила `#1a1a18`, акцент `#c2542f` |
| Палитра (тёмная) | Ночной фон `#15140f`, кремовый текст `#ece7db`, акцент `#df6f44` |
| Текстура | SVG paper grain через `mix-blend-mode: multiply / screen` |
| Курсор | Кастомный lerp-анимированный (только на десктопе) |
| Колонка редактора | 66ch — стандарт iA Writer |
| Сетка | CSS Grid, без UI-фреймворков |

## Технический стек

Приложение намеренно не имеет шага сборки — JSX транспилируется в браузере через Babel standalone.

| | |
|---|---|
| React 18.3.1 | UMD с CDN |
| Babel 7.29.0 | In-browser JSX |
| Хранилище | `localStorage` — ключ `writed:v1` |
| Деплой | GitHub Actions → GitHub Pages |

## Запуск локально

```bash
# из этой папки
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000`.

> Открывать `index.html` напрямую через `file://` не получится — браузер блокирует загрузку модулей. Нужен HTTP-сервер.

## Структура файлов

```
writed./
├── index.html              # Точка входа, подключает React и Babel
├── app.jsx                 # Корень приложения, роутинг, сплеш, курсор
├── store.js                # Стор: localStorage, CRUD, статистика, бэкап
├── components.jsx          # Иконки, Logo, ThemeToggle, хуки, утилиты
├── screens-onboarding.jsx  # 3-шаговый онбординг (имя → тема)
├── screens-dashboard.jsx   # Главная: список проектов, заметок, TopBar
├── screens-editor.jsx      # Редактор с тулбаром, фокус-режимом, превью
├── screens-export.jsx      # Экспорт книги с настройками
├── screens-profile.jsx     # Профиль: аватар, имя, тема, бэкап
├── styles.css              # Дизайн-система: переменные, шрифты, темы
├── ui.css                  # Компонентные стили и адаптив
└── paper.svg               # SVG-текстура бумаги
```
